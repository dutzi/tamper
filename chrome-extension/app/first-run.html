To complete the installation please follow these steps:

  1. If you haven't already, install Tamper's mitmproxy extension

    pip install tamper

  2. Install mitmproxy's certificate. For mitmproxy to inspect and modify packets served behind SSL you will need to install its certificate. Follow the instruction here to complete this step. Make sure Tamper is enabled before visiting mitm.it.

  3. Choose a default editor. When you click on one of the network requests in Tamper's devtools panel, Tamper will try to open that file with the default application for that file type. If your HTML files are opened in Chrome, for example, that file will open in Chrome. You will probably want to change that to some text editor instead. To do that, open the devtools, go to the Tamper panel, click on the cog icon and set the Editor Command Line.
